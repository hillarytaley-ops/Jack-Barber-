APPOINTMENT REMINDERS — Jack's Barber Style
===========================================

Clients get reminders when a booking is set to **Confirmed** in Staff area:
- **24 hours before** the appointment (email + optional SMS)
- **1 hour before** the appointment (email + optional SMS)

STEP 1 — EMAIL (required for reminders and tax invoices)
----------------------------------------------
1. Create a free account at https://resend.com
2. Copy your **API Key**
3. In Vercel → jack-barber → **Settings** → **Environment Variables**, add:
   - Name: RESEND_API_KEY
   - Value: your Resend API key
   - Environment: Production ✓

4. Optional — custom sender (after verifying your domain in Resend):
   - Name: REMINDER_FROM_EMAIL
   - Value: Jack's Barber Style <bookings@yourdomain.com>

   Without this, Resend uses onboarding@resend.dev (fine for testing).

STEP 2 — SMS (optional, paid Twilio account)
--------------------------------------------
1. Create account at https://twilio.com
2. Buy an Australian SMS-capable phone number
3. Add these in Vercel Environment Variables:
   - TWILIO_ACCOUNT_SID
   - TWILIO_AUTH_TOKEN
   - TWILIO_PHONE_NUMBER  (e.g. +614XXXXXXXX)

STEP 3 — CRON SCHEDULE (automatic)
----------------------------------
The website runs reminder checks every 15 minutes (configured in vercel.json).
No extra setup needed once RESEND_API_KEY is set.

STEP 4 — CRON SECURITY (recommended)
------------------------------------
1. In Vercel → Environment Variables, add:
   - Name: CRON_SECRET
   - Value: any long random password you make up
   - Environment: Production ✓

2. Redeploy the site after adding variables.

HOW IT WORKS
------------
- Vercel runs a check every 15 minutes
- Only **Confirmed** bookings get reminders (not Pending or Cancelled)
- After payment, open Staff area → Client Bookings → change status to **Confirmed**
- Reminders use Melbourne time (Wodonga)
