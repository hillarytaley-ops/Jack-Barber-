LIVE WEBSITE STORAGE (one-time setup for bookings + staff area)
===============================================================

Online bookings need shared storage on Vercel so the staff area can see them.

Quick setup (~2 minutes):
1. https://vercel.com/dashboard → open project "jack-barber-rho"
2. Click **Storage** tab
3. Click **Create Database** → choose **KV** (recommended) or **Blob**
4. Click **Connect to Project** → select this website → Connect
5. **Redeploy** (Deployments → latest → Redeploy)

After redeploy:
- Customers can book on the website
- Bookings appear in Staff area → Client Bookings

Local PC (START-SERVER.bat) works without this step.
